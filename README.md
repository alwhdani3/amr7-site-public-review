# AMR7 Public Review Snapshot

هذه نسخة مراجعة عامة آمنة نسبيًا لمشروع amr-7.sa.

## الغرض
رفع ملفات مهمة فقط إلى مستودع GitHub عام مؤقت حتى تتم مراجعتها بواسطة Gemini أو أدوات تحليل خارجية.

## تم استبعاد
- ملفات البيئة والمفاتيح `.env`
- مجلدات `vendor`, `node_modules`, `.git`
- ملفات التخزين العامة `public/storage`
- مرفقات العملاء، الإيصالات، التذاكر، الشركات، ملفات Livewire المؤقتة
- ملفات build/vendor/cache والخرائط والنسخ الاحتياطية

## اطلب من Gemini
راجع هذا المشروع من ناحية Laravel/Livewire, UI/UX RTL, SEO, Security, Performance.
لا تقترح إعادة بناء كاملة. ركز على المشاكل القابلة للإصلاح وأسماء الملفات.
